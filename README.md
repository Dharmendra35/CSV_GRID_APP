# CSV Grid Application

A web application for uploading and analyzing CSV files with sorting, filtering, and conditional styling.

## Features

- CSV file upload (drag & drop or file select)
- Column selection
- Sortable columns
- Advanced filtering (text, number, date)
- Conditional cell styling
- Pagination

## Tech Stack

- React 18
- TypeScript
- AG Grid
- Tailwind CSS
- PapaParse
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
